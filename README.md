# course-Android
Wear

>本课程为Android Wear应用程序开发课程，主要介绍Wear设备与手机设备的互连，Wear设备项目的开发应用，以及Wear开发中的常见问题，通过本课程的学习学生应该能够Wear开发应用的步骤及途径。
本课程是智能设备方向课程体系的高级应用课程。
